# Projeto-Cisc.IA
Criar uma IA que fique em segundo plano, tanto em computadores, celulares, tablet, etc. A IA serviria para pessoas que não sabem mexer em aparelhos e consigam utilizar sem nenhuma dificuldade, podendo ser explicado como usar por voz, texto ou até mesmo setas indicando o quê fazer, tendo acessibilidade para qualquer tipo de pessoa.
